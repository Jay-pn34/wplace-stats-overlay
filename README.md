# Wplace Stats Overlay

A lightweight **Tampermonkey userscript** for [Wplace.live](https://wplace.live/) that shows your **username, level, droplets, and pixels remaining to next level** in a clean overlay.

## Features
- White-themed overlay
- Bold, readable text
- Shows username, current level, droplets, and remaining pixels to next level
- Minimal and compact, positioned at the top-left of the screen
